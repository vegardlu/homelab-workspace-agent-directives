
---
description: Add, commit, and push changes to the .agent repository
---

To update the .agent repository with the latest changes:

1. Stage all changes in the `.agent` directory:
   ```bash
   git add .
   ```

2. Commit the changes with a message:
   ```bash
   git commit -m "update agent configuration"
   ```

3. Push the changes to the remote repository:
   ```bash
   git push
   ```
